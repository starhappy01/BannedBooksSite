# United States Banned Book Map

An interactive map visualization of banned books across the United States, built with React, TypeScript, and Leaflet.

## Features

- Interactive map showing banned books by state
- Year filter slider to view bans by year
- Heatmap coloring based on number of banned books
- Detailed book information panel
- Data sourced from [PEN America](https://pen.org/book-bans/)

## Development

### Prerequisites

- Node.js (v16 or higher)
- npm

### Installation

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

## Data Sources

- **Map Library**: [Leaflet](https://leafletjs.com/)
- **Book Ban Data**: [PEN America](https://pen.org/book-bans/)
